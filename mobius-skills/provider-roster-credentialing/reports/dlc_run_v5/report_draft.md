## 1. Methodology

This report utilizes an outside-in analytical approach, drawing insights from publicly available data and Mobius' proprietary analytical tools. We do not have access to David Lawrence Center's internal HR or credentialing records.

**Methodology Steps:**

1.  **Organizational Identification**: Identified the David Lawrence Center and associated entities through NPPES data, using multiple NPIs linked to the organization's name.
2.  **Location Mapping**: Pinpointed primary service locations using organizational NPI practice addresses from NPPES and validated against service site data (DOGE claims where available).
3.  **Provider Association**: Linked individual NPIs to specific locations based on address matching (from NPPES and roster data) and historical billing patterns.
4.  **Service Portfolio Analysis**: Identified the range of services (taxonomies) provided at each location, cross-referencing with Medicaid approval status (PML/TML).
5.  **Historic Billing Pattern Review**: Analyzed anonymized Medicaid claims data to understand the volume and value of services historically billed by provider types and HCPCS codes.
6.  **Medicaid Enrollment Validation**: Performed four critical checks for each NPI × taxonomy × ZIP+4 combination:
    *   NPI presence in the Provider Master List (PML) with a Medicaid ID.
    *   Validation of the 9-digit ZIP+4 code.
    *   Verification of the taxonomy code against the Taxonomy Master List (TML) or Pending Provider List (PPL).
    *   Confirmation that the NPI+taxonomy+ZIP9 combination has a valid Medicaid ID in the PML.
7.  **Opportunity Sizing**: Quantified potential revenue opportunities by comparing currently valid (A) providers with those flagged for address gaps (B), missing enrollment (C), taxonomy optimization (D), and rate discrepancies (E), using benchmark rates.
8.  **Data Harmonization and Reporting**: Consolidated findings from various sources, applied confidence bands, and structured the data into the waterfall model presented in this report.

**Confidence bands table:**

|---|---|---|---|

**Rate gap methodology note:**
"Section E rate gap figures are calculated by comparing each organization's average paid rate per claim against state-wide average paid rates for the same HCPCS code. This comparison has important limitations: (1) Per diem codes (e.g., H0040 Assertive Community Treatment, S9485 Crisis Intervention) bundle multiple contacts into a single daily charge — their per-claim rates are not directly comparable to fee-for-service encounter rates that other billers may use for the same code. (2) State averages include all biller types; if an organization uses a different service model than the state average, the gap may reflect model differences rather than rate negotiation opportunities. For these reasons, per diem code gaps are shown for informational purposes and should be verified before inclusion in any revenue projection."
